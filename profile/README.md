### 🌐 Overview
**SentryBeacon** là hệ thống giám sát giao thông thông minh sử dụng **AI & Computer Vision**. Chúng tôi tự động hóa việc nhận diện phương tiện để đảm bảo an toàn đường bộ và giảm thiểu sai sót con người.

### 🚀 Key Capabilities
* **Vehicle Detection:** Nhận diện Ô tô, xe máy, xe tải... Real-time.
* **LPR System:** Nhận diện biển số và OCR chính xác cao.
* **Flow Analysis:** Phân tích mật độ và lưu lượng giao thông.
* **Safety Alert:** Cảnh báo vi phạm và nguy cơ mất an toàn.

### 🛠️ Tech Stack
| Core Tech | Tools & Libraries |
| :--- | :--- |
| **Languages** | Python, C++ |
| **AI/ML** | YOLO, PyTorch, TensorFlow |
| **Vision** | OpenCV, IoU Analysis, OCR |
| **System** | Flask Dashboard, Jupyter, Git LFS |

### 🧠 Core Concepts
1.  **Object Detection:** Định vị phương tiện qua Bounding Box (YOLO).
2.  **Segmentation:** Tách ký tự biển số, loại bỏ nhiễu ảnh.
3.  **Optimization:** Dùng **IoU** để loại bỏ vùng nhận diện trùng lặp.
